# “新女性”的建构与再现：以“娜拉”在中国为线索（1900s—1930s）

一个围绕“娜拉”与“新女性”的近代中国妇女与性别史专题资料库。底层逐字保存文献原文，上层叠加史料层级、文献类型、语种、年代、主题、课程讲次等多重标签，并提供一个**双击即开、无需联网**的检索网页，支持多维筛选与组合检索。

> 课程：中国近现代妇女与性别史专题（HIST60116.01，陈雁教授）

## 在线检索

**🔗 https://miniicyfish.github.io/nora-in-china/**

或下载本仓库后，直接双击 `index.html` 在任意浏览器中打开（离线可用）。

## 资料库概况

- 全库 **594 条**：一手史料 439 条（含自脚注辑录的报刊单篇文献 418 篇）、学术研究 151 条、网络资源 4 条。
- 著录统一依《历史研究》大陆式规范（作者：《书名》），原文逐字保留，存疑从阙、绝不臆造。
- 六组分面筛选（史料层级 / 文献类型 / 语种 / 年代分期 / 主题标签 / 课程讲次），组内为“或”、组间为“且”，附全文检索与按年份排序。

## 目录结构

```
.
├── index.html          检索网页（内嵌全库数据的单文件静态页）
├── deliverables/       交付文档
│   ├── ①_资料文献书目.md / .docx      导言 + 分类书目 + 重点文献解题 + 统计概览
│   └── ②_资料库技术规划书.md / .docx
├── data/               数据源与中间产物
│   ├── bibliography.json   全库单一数据源（594 条）
│   ├── paper_articles.json 脚注辑录的报刊单篇文献
│   ├── supplement.json     经核验补入的核心论著
│   └── _*.md               书目正文与统计的中间片段
└── scripts/            生成脚本
    ├── extract.py / extract_notes.py   解析与脚注辑录
    ├── gen_bib.py                      生成书目正文
    ├── build_web.py                    生成检索网页
    └── make_docx.py                    生成 Word 交付文档
```

## 资料来源

资料主体出自许慧琦《“娜拉”在中国》一书所附参考书目；报刊单篇文献则从相关研究论著的征引脚注中爬梳辑录。源 EPUB 受版权所限不随仓库分发；`bibliography.json` 为可直接使用的成品数据，检索网页与书目文档均由其再生。
